# House Styles

This file defines the reusable visual directions that Claude should work within.
These are not one-off mockup styles. They are repeatable product modes.

## Style 1: Soft Premium B2B

Use when:
- the app is business-facing
- the product should feel modern and trustworthy
- the UI needs polish without becoming flashy

Keywords:
- airy
- premium
- rounded
- calm
- warm

Core traits:
- soft rounded cards
- restrained atmospheric gradients or glow
- spacious rhythm
- calm hierarchy
- side panel navigation instead of top tabs
- subtle visual depth rather than heavy decoration

Avoid:
- decorative photography as a primary layout device
- over-branded hero sections in utility screens
- excessive contrast or too many accent colors

Reference anchor:
- `screenshots/Crextio.png`

## Style 2: Premium Utility Lists

Use when:
- the screen is list-heavy
- the user needs to scan, compare, and act quickly
- the interface should feel modern but still highly functional

Keywords:
- list
- premium
- rounded
- minimal
- modern

Core traits:
- clean list rows
- compact, controlled buttons
- typography contrast for hierarchy
- premium rounded containers
- side panel navigation instead of top tabs

Avoid:
- noisy table chrome
- too many action buttons fighting for attention
- generic admin-table styling with weak hierarchy

Reference anchor:
- `screenshots/salesforce.webp`
