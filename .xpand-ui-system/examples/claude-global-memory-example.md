# Example User-Level CLAUDE.md

Use this only after you are happy with the system.

```md
When I ask for UI, UX, styling, screen refactors, component polish, or redesign work, use `.xpand-ui-system/CLAUDE-CODE-ENTRYPOINT.md` as the default UI instruction file.

Before changing UI, read:
- `.xpand-ui-system/CLAUDE-CODE-ENTRYPOINT.md`
- the relevant files it references

Default UI behavior:
- prioritize clarity, consistency, and reusability
- preserve the existing project stack unless I explicitly ask to change it
- design for desktop and mobile
- cover loading, empty, error, hover, focus, and disabled states when relevant
- keep accessibility in scope
- prefer reusable patterns over one-off decorative changes
```
